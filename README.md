# Scientific Visualization Toolkits (SciViz-Toolkit)

A Python toolkit for generating publication-quality scientific visualizations. Optimized for academic journals with standard color palettes and high-resolution exports.

一个用于生成出版级科研可视化的 Python 工具包。针对学术期刊进行了优化，支持标准色板和高分辨率导出。

-----

## 🌟 简介 (Introduction)

**CN:** 本项目旨在解决科研绘图中“默认样式不好看、调整参数太麻烦”的痛点。通过封装复杂的绘图逻辑，你可以一键生成符合 Nature/Science/Cell 等顶级期刊格式要求的矢量图表。

**EN:** This project aims to solve the pain points in scientific plotting, such as unattractive default styles and tedious parameter adjustments. By encapsulating complex plotting logic, you can generate vector charts that meet the formatting requirements of top-tier journals like *Nature*, *Science*, and *Cell* with just one click.

-----

## ✨ 核心特性 (Key Features)

  * **🎨 Journal-Ready Themes:** Preset color palettes and styles for Nature, Science, and Cell.  
    **期刊级主题：** 预设 Nature, Science, Cell 等顶级期刊的配色方案与样式。
  * **📐 High Resolution:** Supports 800+ DPI export and vector formats including `.pdf`
    **高分辨率：** 默认支持 800+ DPI 导出，支持 `.pdf`, `.svg`, `.eps` 等矢量格式。
  * **🐍 Ease of Use:** Seamless integration with Pandas and NumPy; generate plots in just a few lines of code.  
    **易于使用：** 完美兼容 Pandas 和 NumPy 数据格式，仅需几行代码即可出图。
  * **🤖 AI-Assisted:** Developed with the assistance of AI, ensuring clean code and modern implementation.  
    **AI 辅助开发：** 本项目由 AI 协作完成，确保了代码的简洁与现代化的实现。

## 🚀 快速上手 (Quick Start)

### Installation / 安装

```bash
git clone https://github.com/你的用户名/scientific-visualization-toolkits.git
cd scientific-visualization-toolkits
pip install -r requirements.txt
```
